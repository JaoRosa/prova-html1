Parte 1 – Teórica (7 questões)

  1. (1,0 pt) Explique com suas palavras o que é a Internet e como ela surgiu.

    2. R:A internet e o meio que transporta todo e qualquer tipo de informação criada digitalmente pelo homem ate hoje com sua principal característica sendo a sua conexão via satélite ou a cabo.

  2. (1,0 pt) O que é o HTML? Por que ele é importante para o desenvolvimento de páginas web?

    3. R:HTML (HyperText Markup Language) é a linguagem de marcação padrão ele é importante porque fornece a estrutura básica de todo site deixando possivel que os navegadores interpretem e mostrem texto, imagens e outros elementos de forma organizada e acessível

  3. (1,0 pt) Qual a diferença entre as tags < ol > e < ul > no HTML? Dê um exemplo de cada

    4. R: < ol >: Ordered list – lista ordenada em números (própria para passo a passo de um tutorial) < ul >: Unordered list – lista não ordenada (própria para lista d itens para levar em uma viagem)

  4. (1,0 pt) Para que serve a tag no HTML? Mostre um exemplo funcional

    5. R:< a >: usado para fazer ancora com um link através de uma palavra

  5. (1,0 pt) Explique a importância da semântica em HTML5. Cite ao menos duas tags semânticas e seus significados

    6. R: A semântica é importante por que permite um melhor entendimento e organização da pag e assim também melhorando a acessibilidade para a organização e a busca e a manutenção futura do site

    O < header > contém o título da página e um menu de navegação.

    O < footer > direitos autorais e informações de contato caracteristicas comuns de fim de pag

  6. (1,0 pt) Qual é a função do atributo alt na tag < img >? Por que ele é essencial para acessibilidade?

    7. R:a função serve para adicionar um texto para que quando o site for interpretado para pessoas deficientes visuais a imagem possa ser lida como texto
